# ES6 Data Manipulation

This project contains JavaScript exercises for ES6 data manipulation.

## Tasks

### 0. Basic list of objects
Create a function `getListStudents` that returns an array of objects.

Each object has three attributes:
- id (Number)
- firstName (String)
- location (String)

### 1. More mapping
Create a function `getListStudentIds` that returns an array of student ids.

### 2. Filter
Create a function `getStudentsByLocation` that returns students by location.

### 3. Reduce
Create a function `getStudentIdsSum` that returns the sum of all student ids.

### 4. Combine
Create a function `updateStudentGradeByCity` that updates student grades by city.
